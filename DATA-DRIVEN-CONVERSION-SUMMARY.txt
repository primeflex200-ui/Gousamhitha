HOW TO USE:
====================================================

STEP 1: SETUP SUPABASE
1. Create Supabase project
2. Run supabase-schema.sql in SQL Editor
3. Create storage buckets
4. Copy API credentials to backend/.env

STEP 2: CLEAR DEMO DATA
1. Open clear-demo-data.html in browser
2. Click "Clear All Data"
3. Confirm deletion

STEP 3: ADD FILES TO HTML PAGES
Add to ALL HTML pages (before closing </body>):

<link rel="stylesheet" href="empty-states.css">
<script src="data-manager.js"></script>

STEP 4: START FRESH
1. Login as admin: admin@cb.com / admin123
2. Go to Admin Dashboard
3. Add first product
4. Product appears in shop
